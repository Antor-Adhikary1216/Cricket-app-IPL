# 🏏 Cricket App IPL

A modern IPL cricket player selection web app built with **React**, **Vite**, **Tailwind CSS**, and **DaisyUI**.
The app displays available cricket players with their country, role, rating, batting style, bowling style, and price in a clean card-based UI.

## ⚙️ Installation and Setup

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/Antor-Adhikary1216/Cricket-app-IPL.git
```

### 2. Go to the project folder

```bash
cd Cricket-app-IPL
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

Now open the local server link in your browser.

## 🧠 How the App Works

The app loads player data from the `players.json` file and displays each player using a reusable `PlayersCard` component.

Each player card includes:

* Player image
* Player name
* Country
* Role
* Rating
* Batting style
* Bowling style
* Price
* Choose Player button

The main app also includes an **Available / Selected** tab system to switch between available players and selected players UI.
